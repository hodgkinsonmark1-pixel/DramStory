# Brand Voice — How to Actually Write for This Site

Extracted from the original DramStory operating guide (superseded 19 July
2026 - this and `content-structure-conventions.md` between them cover
everything from that guide not already living in another doc).

- **Tone:** insider, playful, sophisticated. Celebrates *planning* as part
  of the adventure, not just the destination.
- **Write like someone who's actually been there**, not like marketing copy
  describing a place secondhand. Concrete, specific, sensory details beat
  generic superlatives every time - "the pool is heated by waste heat piped
  straight from Bowmore Distillery next door" beats "a unique local
  attraction."
- **Evocative but not overwrought.** Short paragraphs, real texture, no
  purple prose. A day's narrative should set a mood and explain *why* the
  day feels the way it does - it is not a restatement of the stop list
  underneath it. See `day-narrative-standard.md` for the fuller, Hub-Day-
  specific version of this same principle, with worked before/after
  examples.
- **Honesty over salesmanship.** If something needs booking ahead, say so
  plainly. If accommodation isn't bookable yet, say that too, rather than
  implying more polish than exists. The brand's credibility depends on this
  - it's a planning tool for people who'll actually show up at these
  places.
- **Never fabricate specific factual claims** (opening dates, prices,
  historical trivia) that aren't grounded in verified source data. General
  emotive description of a well-known, visually obvious place (e.g.
  describing a beach as beautiful) is fine; inventing unverifiable
  specifics is not.
